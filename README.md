# ai-ify

## Cursor Skills

The custom Cursor skills in the `skills/` directory are the canonical source for this repository. They are designed to be symlinked to `~/.cursor/skills/` on a machine bootstrap.

- [`skills/planify`](skills/planify)
- [`skills/buildify`](skills/buildify)
- [`skills/verifyify`](skills/verifyify)
- [`skills/specify`](skills/specify)

Builtin skills in `~/.cursor/skills-cursor` and plugin-managed skills are intentionally excluded from this repository as they are managed by Cursor itself.
